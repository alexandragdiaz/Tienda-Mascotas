# PetStore 🐾

PetStore es una tienda virtual para mascotas desarrollada con HTML, CSS y JavaScript.  
El proyecto cuenta con un diseño moderno, responsive y una interfaz enfocada en mostrar productos para mascotas de forma atractiva y dinámica.

## 🚀 Características

- Diseño responsive para móviles y escritorio
- Catálogo de productos dinámico
- Menú de navegación adaptable
- Botón flotante de WhatsApp
- Animaciones suaves al hacer scroll
- Footer moderno con redes sociales
- Interfaz visual moderna y amigable

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome
